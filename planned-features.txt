Dashboard Priorities
- Library Overview: scrollable widget with playlists, expandable to show track rows, or in accompanying "playlist overview"
- Import Control: Button to import playlists. will open up a window prompting use to select import via local files, or the spotify API.
- Import status indicator? Dashboard or import window could show progress as the full set of playlists is imported. CSV import can take up to 10 seconds (but predictable since # of files and progress are easily trackable) and spotify API will take much longer.
- Export Control: Same as import, let user choose export mode: local files, spotify API.
- Workspace launcher: determine playlists selection, likely via popup or expanded window space, opening workspace with selections upon confirmation.
- Workspaces management? possibly track different sets of playlists selected for the workspace, letting the user continue work on a project later.
